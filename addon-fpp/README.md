# FPP - Falcon Player Add-On for Home Assistant

[![FPP logo](https://raw.githubusercontent.com/jessica12ryan/homesync-addons/master/addon-fpp/logo.png)](https://github.com/FalconChristmas/fpp/)

## Getting Started

- WARNING: THIS ADDON IS FOR ADVANCED USERS ONLY!
- AS SOON AS YOU STOP THE ADDON, IT WILL LOSE ALL DATA. THIS ADDON WAS CREATED FOR TESTING AND DEVELOPMENT!

- After clicking install, it may take several minutes to download and compile FPP. If you refresh or browse to another page, the addon will continue downloading. The addon page will update when the addon has been successfully installed.
- You can open FPP by browsing to http://HA_IP
